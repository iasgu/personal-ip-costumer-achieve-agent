# 舅妈抖音互动评论访谈工具

这是一个轻量 H5 访谈工具，用来把一个常刷抖音用户的评论互动直觉，转成可复用的评论回复规则。

## 目标

你不需要亲自访谈。把链接发给她，她用手机完成：

1. 基础互动习惯问题。
2. 评论分类判断题。
3. 15 个真实评论场景回复题。
4. 负面评论/质疑评论处理题。
5. 私域引导和风险避坑题。

系统输出：

- 评论互动风格
- 评论分类规则
- 高价值线索识别规则
- 负面评论处理规则
- 私域引导边界
- 避坑清单
- 可导入 AI 工作流的 JSON

## 启动

```powershell
cd prototype/auntie-douyin-interview
node server.js
```

打开：

```text
http://127.0.0.1:8894/
```

## 当前版本

- 不依赖数据库。
- 数据保存到 `data/submissions.json`。
- 支持手机浏览器。
- 支持文字回答。
- 语音采集后续可接浏览器 MediaRecorder 或小程序录音。
