# 2026-06-25 西安测试准备清单

## 测试目标

验证「个人 IP 智能体」的可演示闭环：

```text
爆款视频输入 -> 生成口播视频 -> 生成发布草稿 -> 模拟发布 -> 评论互动回复
```

其中发布与评论今天先使用 `publish-interaction-agent` 的 mock/local adapter，等抖音开放平台权限通过后再切换真实 adapter。

当前状态：

- 已申请个人开发者认证。
- 官方视频发布/评论权限预计不会马上通过。

测试策略：

- 明天先演示 mock/local 发布与评论闭环。

真实接入需要：

- `DOUYIN_CLIENT_KEY`
- `DOUYIN_CLIENT_SECRET`
- `DOUYIN_REDIRECT_URI`
- `DOUYIN_ACCESS_TOKEN`
- 视频发布权限
- 评论管理权限

## 演示素材

建议准备：

- 1 张清晰正脸头像。
- 1 段 10-20 秒本人声音。
- 1 条 15-30 秒抖音视频链接。
- 1 个默认个人定位：个人 IP 口语化短视频获客顾问。
- 1 条生成后的视频样片。

## 风险说明

- 抖音官方发布和评论回复不能绕过开放平台权限。
- 评论回复只能回复授权账号自己发布的视频评论。
- 即梦是否优于 Wan，需要按成本、耗时、质量实测。
- 剪映目前没有看到稳定官方后端 API，暂不作为核心自动化方案。
- 开拍更像口播剪辑工具，适合流程对比，不适合直接做后端核心模块。
